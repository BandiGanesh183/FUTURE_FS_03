Ganesh Events & Decor is a premier event decoration company based in Anantapur, Andhra Pradesh. 
This responsive website serves as a digital portfolio and lead generation platform, showcasing expertise in wedding decor, birthday celebrations, flower arrangements, and balloon decorations. 
The website helps the business establish a strong online presence and connect with customers seeking professional decoration services.

📌 Project Overview
This website helps Ganesh Events & Decor establish a strong online presence, showcase their portfolio, attract customers, and generate leads for wedding, birthday, and event decorations. Built with pure HTML, CSS, and JavaScript – no frameworks, no backend required.
